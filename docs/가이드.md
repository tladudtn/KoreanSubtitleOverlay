# GTA 산안드레아스 한글 자막 오버레이 - 사용 가이드

이 문서는 코드를 몰라도 이해할 수 있도록 쓴 안내서입니다. 개발자용 상세 내용은
[README.md](../README.md)와 [DEVELOPER_GUIDE.md](DEVELOPER_GUIDE.md)를 참고하세요.

## 이게 뭔가요?

GTA 산안드레아스를 커뮤니티 한글패치와 같이 돌리면, 게임 안에서
나오는 대사 자막(미션 목표, 캐릭터 대사 등)의 한글이 삐뚤빼뚤 기울어진
채로 나옵니다. 초성/중성/종성 조각을 억지로 간격을 좁혀 겹쳐 붙여서
완성된 글자처럼 흉내 내는 방식이다 보니, 조각끼리 딱 맞게 정렬되지
않고 비스듬히 기울어진 것처럼 보이는 겁니다.

이 모드는 그 기울어진 자막을 감지해서, 화면 위에 **제대로 된 반듯한
한글로 다시 그려주는** modloader용 플러그인(.asi)입니다.

## 왜 자막이 기울어지나요?

한글패치가 쓰는 글꼴 파일(`fonts.txd`)은 일반적인 "글자 하나 = 코드 하나"
방식이 아니라, 옛날 콘솔 게임에서 쓰던 **자모 조각 방식**입니다. 즉 "가"
같은 완성된 글자 하나를 저장하는 게 아니라, ㄱ(초성) + ㅏ(중성) 조각을
각각 저장해뒀다가(원래는 풀어쓰기로 따로따로 저장된 조각들), 화면에
그릴 때 간격을 좁혀 겹쳐 그려서 모아쓰기처럼 보이게 흉내 내는 방식입니다.

문제는 이게 진짜 글꼴 차원의 조합이 아니라 그냥 조각들을 억지로 가까이
붙여놓은 것뿐이라서, 조각끼리 딱 맞게 정렬되지 않고 삐딱하게 기울어진
채로 보인다는 겁니다.

## 이 모드는 어떻게 문제를 고치나요?

1. 게임이 자막을 화면에 그리기 **직전**에 가로채서, 원래 그리기 기능은
   꺼버립니다 (그래서 기울어진 자막이 안 보입니다).
2. 그 대신 게임 메모리에서 자막 원본 데이터(자모 조각들)를 직접 읽어옵니다.
3. 이 조각들을 올바르게 조합해서 진짜 유니코드 한글 글자로 바꿉니다.
4. 그 결과를 화면 위에 (Windows에 이미 깔려있는 "맑은 고딕" 글꼴로,
   굵게+테두리 있게) 새로 그립니다.

즉, 한글패치의 폰트 파일이나 게임 파일은 전혀 건드리지 않고, "그리는
방식"만 바꿔치기하는 오버레이입니다.

몇 가지 세부 동작도 신경 써서 처리했습니다:
- 게임 옵션에서 "자막"을 꺼놔도, 미션 중 캐릭터끼리 나누는 대사는 항상
  나오도록 자막 설정을 내부적으로 강제로 켭니다.
- MISSION PASSED / BUSTED / WASTED처럼 원래부터 영어인 화면은 이 모드가
  건드리지 않고 게임 본래 모습 그대로 나옵니다.
- ESC를 눌러 일시정지 메뉴를 열면 자막이 자동으로 숨겨지고, 메뉴를 닫으면
  다시 나타납니다.

## 설치 방법

**준비물**
- GTA 산안드레아스 + [modloader](https://gtaforums.com/topic/577721-relmod-modloader/) 설치되어 있을 것
- 한글패치(`fonts.dat`, `fonts.txd`, `american.gxt`)가 이미 적용되어 있을 것
  (이 모드는 자막을 "고쳐 그리는" 것이지, 번역 자체를 추가하지는 않습니다)

**설치 순서**
1. `KoreanSubtitleOverlay.asi` 파일을 받습니다 (직접 빌드하거나 배포본을 받음).
2. GTA SA 설치 폴더 안의
   `modloader\_BASIC\KoreanSubtitleOverlay\KoreanSubtitleOverlay.asi`
   경로에 복사합니다. (`KoreanSubtitleOverlay` 폴더가 없으면 새로 만드세요.)
3. 글자 크기/위치/폰트를 바꾸고 싶다면 `KoreanSubtitleOverlay.ini` 파일
   (커스텀 폰트를 쓸 경우 `fonts` 폴더도)을 같은 폴더에 복사합니다
   (둘 다 선택사항 - 안 넣어도 기본값으로 동작합니다).
4. 게임을 실행합니다.

정상 동작하면 미션 대사나 목표 텍스트가 화면 하단에 흰 글씨 + 검은
테두리로, 배경 상자 없이 깔끔하게 나옵니다.

## 글자 크기/위치/폰트 조정하기

`KoreanSubtitleOverlay.ini` 파일(메모장으로 열면 됩니다)에서 다음 값을
바꿀 수 있습니다. 바꾼 뒤에는 게임을 완전히 껐다가 다시 켜야 적용됩니다.

```ini
[Subtitle]
FontSize=39   ; 자막 글자 크기. 숫자를 키우면 글자가 커집니다.
OffsetX=0     ; 자막을 화면 중앙 기준 오른쪽(+)/왼쪽(-)으로 옮깁니다.
OffsetY=0     ; 자막을 아래(+)/위(-)로 옮깁니다.
Debug=no      ; 문제 진단용 상세 로그. 평소엔 no로 두세요.
FontPath=     ; 비워두면 시스템 맑은 고딕 사용. 다른 폰트를 쓰려면
              ; fonts 폴더에 파일을 넣고 FontPath=fonts/MyFont.ttf
              ; 처럼 지정하세요 (자세한 건 fonts 폴더 안 안내 참고).
```

## 뭔가 안 될 때

- `.asi` 파일과 같은 폴더에 `KoreanSubtitleOverlay.log`라는 로그 파일이
  자동으로 생깁니다. 문제가 있으면 이 파일에 원인이 적혀 있을 가능성이
  높습니다.
- 어떤 대사가 왜 이상하게 나오는지 더 자세히 알고 싶다면
  `KoreanSubtitleOverlay.ini`에서 `Debug=yes`로 바꾸고 게임을 재시작하면
  로그에 훨씬 자세한 정보가 남습니다 (평소엔 로그가 커지니 다시 `no`로
  돌려놓는 걸 권장합니다).
- 게임을 완전히 껐다가 다시 켜야 새로 설치한 `.asi`가 적용됩니다. (미션
  재시작만으로는 적용 안 됩니다.)
- 한글패치 자체가 안 깔려 있으면 이 모드는 아무 효과가 없습니다.

## 자주 묻는 질문

**Q. 미션 클리어/체포/사망 화면(MISSION PASSED, BUSTED, WASTED)도 이 모드가
그리나요?**
아니요. 그 배너들은 원래 영어라서 게임 기본 글꼴로도 멀쩡하게 나오기
때문에, 이 모드는 한글이 섞인 대사에만 관여하고 저런 배너는 그대로
둡니다.

**Q. 컷신이 아닌 일반 플레이 중 대사도 나오나요?**
네. 컷신 여부와 상관없이, 게임이 화면에 띄우려는 대사가 있으면 전부
감지해서 다시 그립니다.

**Q. 아직도 가끔 이상한 글자가 섞여 나와요.**
이 모드는 한글패치가 쓰는 독자적인 자모 조각 인코딩을 통째로
역공학해서 만든 것이라, 아직 확인 안 된 조합이 발견될 수 있습니다.
어떤 문장이 어떻게 깨졌는지(스크린샷 등) 알려주시면 원인을 찾아
고칠 수 있습니다.
